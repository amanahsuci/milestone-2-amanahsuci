# Suci's Project 

## Milestone-2 

### 🎮 Game Page by revofun.inc

#### Welcome to playgamedulzz.inc Game Page by **Amanah Suci**. An interactive game page which can fill your free time. This page has 3 game that you can play!

### 1. Guessing Number Game 🔢
#### Guess a number between 1 and 100 in max- 5 chances
**Feature** :
- Number input and guess input buttons for player
- Message display for game result
- Step Counter
- Restart, stop, and back to Home buttons

### 2. Rock Paper Scissors ✋🏻✊🏻✌🏻
#### A classic game here! Rock, paper, and scissors game for player 'vs' computer. 
**Feature**
- An animation/image hand for player 'vs' computer
- Rock, paper, and scissors buttons for player
- Message display for game result
- Restart, stop, and back to Home buttons

### 3. Memory Card Game 🃏
#### Match the cards!
**Feature** 
- 8 pairs of random cards
- Flip cards animations
- The cards remain open if they are matched
- Step display

## Technologies Used
- HTML for web structures
- CSS for styling the web
- Javascript for interactive games
